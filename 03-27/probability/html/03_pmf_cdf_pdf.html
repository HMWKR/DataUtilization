1. 확률질량함수 (PMF)란?

확률질량함수(Probability Mass Function, PMF)는 이산 확률변수가 특정 값을 가질 확률을 나타내는 함수입니다. 쉽게 말해, "이 값이 나올 확률이 얼마인가?"를 알려주는 함수입니다.

$$P(X = k) = \text{(확률변수 X가 정확히 k일 확률)}$$

용어 정의

핵심 용어

확률질량함수 (PMF) 이산 확률변수의 각 값에 확률을 대응시키는 함수. "질량(mass)"은 확률이 특정 점에 집중되어 있다는 의미입니다.
이산 확률변수 셀 수 있는 값만 가지는 확률변수 (예: 주사위 눈의 수, 동전의 앞/뒤). 연속적이지 않고 "띄엄띄엄" 존재합니다.
PMF의 성질 모든 확률값은 0 이상이고, 모든 확률의 합은 정확히 1입니다: $$\sum_{k} P(X=k) = 1$$

왜 합=7의 확률이 가장 높을까?

합이 7이 되는 조합이 6가지로 가장 많기 때문입니다: (1,6), (2,5), (3,4), (4,3), (5,2), (6,1)

반면 합이 2가 되는 조합은 (1,1) 딱 1가지뿐입니다. 36개의 동일한 결과 중 해당 조합이 차지하는 비율이 곧 확률입니다.

  • P(X=7) = 6/36 = 0.167 (가장 높음)
  • P(X=2) = P(X=12) = 1/36 = 0.028 (가장 낮음)

이 분포는 좌우 대칭이며, 합 7을 중심으로 삼각형 모양을 이룹니다.

2. 누적분포함수 (CDF)란?

누적분포함수(Cumulative Distribution Function, CDF)는 확률변수가 특정 값 이하일 확률을 나타냅니다. PMF가 "정확히 이 값일 확률"이라면, CDF는 "이 값까지의 확률을 다 합한 것"입니다.

$$F(x) = P(X \leq x) = \sum_{k \leq x} P(X = k)$$

용어 정의

핵심 용어

누적분포함수 (CDF) 확률변수 X가 x 이하일 확률. PMF를 왼쪽에서부터 차례로 더한 것입니다.
누적확률 특정 값까지 확률을 모두 합한 값. CDF는 항상 0에서 시작해서 1로 끝납니다.
CDF의 성질 단조증가 (절대 감소하지 않음), $$\lim_{x \to -\infty} F(x) = 0, \quad \lim_{x \to +\infty} F(x) = 1$$

PMF에서 CDF로 변환

PMF의 각 확률값을 왼쪽부터 순서대로 누적하면 CDF가 됩니다. 이산형에서 CDF는 계단 함수 모양을 합니다.

$$F(5) = P(X \leq 5) = P(X=2) + P(X=3) + P(X=4) + P(X=5) = \frac{1+2+3+4}{36} = \frac{10}{36} \approx 0.278$$

CDF의 계단 모양은 무엇을 의미할까?

이산형 CDF의 수직 점프 크기가 바로 해당 값의 PMF(확률)입니다.

예를 들어 X=7에서의 점프 크기 = P(X=7) = 6/36 = 0.167이며, 이것은 PMF 막대의 높이와 동일합니다.

  • 점프가 큰 곳 = 해당 값의 확률이 높음
  • 수평 구간 = 해당 범위에서 확률변수가 값을 갖지 않음
  • 최종 높이 = 1 = 모든 확률의 합

3. 확률밀도함수 (PDF)란?

확률밀도함수(Probability Density Function, PDF)는 연속 확률변수의 분포를 나타내는 함수입니다. PMF와 달리, PDF에서 특정 점의 확률은 항상 0입니다. 확률은 구간의 면적으로만 구할 수 있습니다.

$$P(a \leq X \leq b) = \int_a^b f(x)\,dx = \text{(곡선 아래 면적)}$$

용어 정의

핵심 용어

확률밀도함수 (PDF) 연속 확률변수의 분포를 나타내는 함수. y축 값은 "밀도"이지 확률이 아닙니다.
확률밀도 (Density) 단위 구간당 확률의 집중도. 밀도 자체는 1을 초과할 수 있습니다!
연속 확률변수 셀 수 없이 많은 값을 가지는 확률변수 (예: 키, 몸무게, 시간). 어떤 두 값 사이에도 무한히 많은 값이 존재합니다.

히스토그램에서 PDF로

히스토그램의 bins 수를 늘리면 점점 매끄러운 곡선에 가까워집니다. 이 극한이 바로 PDF입니다.

핵심 질문: "밀도는 확률인가?"

X (틀림)

"y축 값 0.04 = 확률 4%"

밀도를 확률로 착각하면 안 됩니다

O (맞음)

"면적(밀도 x 구간폭) = 확률"

밀도는 확률이 아니라 확률의 "농도"입니다

$$\int_{-\infty}^{+\infty} f(x)\,dx = 1 \quad \text{(전체 면적의 합 = 1)}$$

주의: 밀도를 "더하면" 1이 아닙니다. "적분하면" 1입니다!

면적 = 확률 계산기

아래 정규분포 곡선에서 범위를 조절하면, 해당 구간의 면적(=확률)이 실시간으로 계산됩니다.

표준편차를 2~3 정도로 줄여보세요. 밀도(y축)가 1을 초과하는 것을 확인할 수 있습니다! 이것이 "밀도는 확률이 아니다"의 직관적 증거입니다.

밀도가 1을 초과할 수 있다고? 그러면 확률이 100%가 넘는 거 아닌가?

아닙니다! 확률밀도와 확률은 다른 개념입니다.

비유를 들어 설명하면:

  • 밀도 = 인구 밀도 (1km^2당 몇 명). 인구밀도가 10,000명/km^2여도, 전체 인구의 합이 국가 인구를 넘지 않습니다.
  • 확률 = 면적. 밀도(높이) x 구간 폭 = 면적 = 확률

표준편차가 매우 작은 정규분포의 경우, 좁은 범위에 확률이 집중되므로 밀도값(y축)이 1을 초과합니다. 하지만 전체 곡선 아래 면적은 항상 정확히 1입니다.

$$f(x) > 1 \text{ 가능},\quad \text{하지만} \int_{-\infty}^{+\infty} f(x)\,dx = 1 \text{ 항상 성립}$$

4. KDE (커널 밀도 추정)란?

KDE(Kernel Density Estimation)는 데이터로부터 연속적인 확률밀도함수를 추정하는 방법입니다. 히스토그램의 한계(bins 수에 따라 모양이 달라짐)를 극복하기 위해 사용합니다.

용어 정의

핵심 용어

커널 (Kernel) 각 데이터 포인트에 올려놓는 작은 확률분포 함수 (보통 가우시안). 각 데이터 점 주변의 확률 기여를 나타냅니다.
대역폭 (Bandwidth) 커널의 폭을 결정하는 매개변수. 크면 부드럽고 작으면 울퉁불퉁해집니다.
가우시안 KDE 커널로 정규분포(가우시안)를 사용하는 KDE. seaborn의 kdeplot이 기본적으로 사용합니다.
$$\hat{f}(x) = \frac{1}{nh} \sum_{i=1}^{n} K\left(\frac{x - x_i}{h}\right)$$

n: 데이터 수, h: 대역폭, K: 커널 함수, x_i: 각 데이터 포인트

같은 데이터인데 왜 bins 수에 따라 모양이 달라질까?

히스토그램은 데이터를 구간(bin)으로 나눠서 세는 방식입니다.

  • bins이 너무 적으면: 세부 구조가 사라지고 너무 매끈해짐 (과소적합, underfitting)
  • bins이 너무 많으면: 노이즈까지 반영해서 울퉁불퉁 (과대적합, overfitting)

이 문제를 해결하기 위해 KDE를 사용합니다. KDE는 각 데이터 점에 작은 커널(보통 가우시안)을 올려놓고 합산하여 부드러운 곡선을 만듭니다.

KDE에서도 대역폭(bandwidth) 선택이 중요합니다:

  • 대역폭이 너무 작으면: 각 데이터 점의 형태가 드러남 (과대적합)
  • 대역폭이 너무 크면: 진짜 분포의 세부 구조를 놓침 (과소적합)

왜 KDE를 사용하는가?

히스토그램의 한계

  • bins 수에 따라 모양이 변함
  • bin 경계 위치에 따라 달라짐
  • 불연속적 (계단 모양)
  • 데이터가 적으면 불안정

KDE의 장점

  • 매끄러운 연속 곡선
  • bin 선택 문제 없음
  • 확률밀도함수 직접 추정
  • 시각적으로 직관적
Python에서는 어떻게 사용할까?

seaborn 라이브러리에서 간단하게 사용할 수 있습니다:

sns.histplot(data=titanic, x='age', kde=True)

kde=True 옵션 하나만 추가하면 히스토그램 위에 KDE 곡선이 함께 그려집니다.

KDE만 따로 그리려면: sns.kdeplot(data=titanic, x='age')

대역폭 조절: sns.kdeplot(data=titanic, x='age', bw_adjust=0.5)
