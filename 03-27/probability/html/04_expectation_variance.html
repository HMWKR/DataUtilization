1. 기대값이란?

"평균적으로 기대하는 값" -- 기대값(Expectation, Expected Value)은 확률변수가 취할 수 있는 모든 값을 확률로 가중 평균한 것입니다.

$$E[X] = \sum_{x} x \cdot P(X=x)$$

쉽게 말하면: "이 실험을 무한히 반복하면, 평균적으로 어떤 값이 나올까?"에 대한 답입니다.

왜 "기대"값이라 부르나요?

17세기 도박 문제에서 유래합니다. 도박꾼이 게임에서 "기대"할 수 있는 이익을 계산하기 위해 만들어진 개념입니다.

예를 들어, 동전 던지기에서 앞면이 나오면 100원, 뒷면이면 0원을 받는 게임이라면:

  • E[X] = 100 x 0.5 + 0 x 0.5 = 50원
  • 한 번에 정확히 50원을 받는 것은 아니지만, 평균적으로 50원이라는 뜻

일상 속 기대값

상황확률변수기대값
공정한 주사위 1개X = 나오는 눈E[X] = 3.5
동전 2개 (앞=1, 뒤=0)X = 앞면 수E[X] = 1
로또 1장X = 당첨금E[X] ≈ -500원 (기대 손실)

2. 주사위 기대값 — 밸런스 빔으로 이해

기대값은 분포의 "무게 중심"입니다. 아래 밸런스 빔에서 각 눈의 확률(무게)을 조절하고, 기대값(받침점)이 어떻게 이동하는지 관찰하세요.

밸런스 빔 기대값 시각화

기대값 E[X]
3.50
분산 Var[X]
2.92
표준편차
1.71
왜 주사위 기대값이 3.5일까요?

주사위의 각 눈은 1/6 확률로 나옵니다:

$$E[X] = 1 \times \tfrac{1}{6} + 2 \times \tfrac{1}{6} + 3 \times \tfrac{1}{6} + 4 \times \tfrac{1}{6} + 5 \times \tfrac{1}{6} + 6 \times \tfrac{1}{6} = \frac{21}{6} = 3.5$$

3.5는 주사위에 없는 눈이지만, 여러 번 던졌을 때의 평균은 3.5에 수렴합니다. 이것이 대수의 법칙(Law of Large Numbers)입니다.

3. 분산이란?

"흩어짐의 정도" -- 분산(Variance)은 각 값이 기대값에서 얼마나 떨어져 있는지를 측정합니다.

$$\text{Var}[X] = E[(X - \mu)^2] = \sum_{x}(x - \mu)^2 \cdot P(X=x)$$

분산이 크면 값들이 넓게 퍼져 있고, 작으면 기대값 주변에 몰려 있습니다.

분산의 직관

  • 편차(deviation): 각 값에서 평균을 뺀 것 $(x - \mu)$
  • 제곱하는 이유: 양수/음수가 상쇄되는 것을 막기 위해
  • 가중 평균: 확률로 가중하여 평균 내면 -> 분산
  • 표준편차 $\sigma$: 분산에 루트를 씌우면 원래 단위로 복원

분산 링(Variance Ring) 비교 시각화

두 분포의 분산을 비교합니다. 링의 반지름이 표준편차를 나타냅니다.

분포 A
3.50
Var=2.92, SD=1.71
분포 B
5.17
Var=1.14, SD=1.07
왜 편차를 "제곱"할까요? 절댓값은 안 되나요?

수학적으로 절댓값도 가능합니다 (그것이 MAD, Mean Absolute Deviation). 하지만 제곱을 쓰는 이유:

  • 미분 가능성: 제곱 함수는 모든 점에서 미분 가능 (절댓값은 0에서 불가)
  • 수학적 편의: 분산의 성질 $\text{Var}[aX+b] = a^2\text{Var}[X]$이 깔끔
  • 정규분포와의 자연스러운 연결: 정규분포의 PDF에 $\sigma^2$이 등장

ML에서는 MSE(Mean Squared Error)가 분산 개념의 직접적 확장입니다.

4. 이산 vs 연속 — 합(Σ) vs 적분(∫)

이산형과 연속형은 같은 개념을 다른 도구로 계산합니다. 합(Σ)이 적분(∫)으로, 확률질량함수(PMF)가 확률밀도함수(PDF)로 바뀔 뿐입니다.

이산 vs 연속 수식 비교

이산형 (Discrete)

확률질량함수 PMF: $P(X=x)$

$$E[X] = \sum_{x} x \cdot P(X=x)$$
$$\text{Var}[X] = \sum_{x}(x-\mu)^2 P(X=x)$$

예: 주사위, 동전, 카드

연속형 (Continuous)

확률밀도함수 PDF: $f(x)$

$$E[X] = \int_{-\infty}^{\infty} x \cdot f(x)\,dx$$
$$\text{Var}[X] = \int_{-\infty}^{\infty}(x-\mu)^2 f(x)\,dx$$

예: 키, 몸무게, 온도

이산형 확률변수 (Discrete Random Variable)

셀 수 있는 값을 가지는 확률변수

$$E[X] = \sum_{i=1}^{n} x_i \cdot P(X=x_i)$$
$$\text{Var}[X] = \sum_{i=1}^{n}(x_i - \mu)^2 \cdot P(X=x_i) = E[X^2] - (E[X])^2$$
$x$123456
$P(X=x)$$\frac{1}{6}$$\frac{1}{6}$$\frac{1}{6}$$\frac{1}{6}$$\frac{1}{6}$$\frac{1}{6}$
연속형 확률변수 (Continuous Random Variable)

연속적인 구간의 값을 가지는 확률변수

$$E[X] = \int_{-\infty}^{\infty} x \cdot f(x)\,dx$$
$$\text{Var}[X] = \int_{-\infty}^{\infty}(x-\mu)^2 f(x)\,dx = E[X^2] - (E[X])^2$$

연속형에서는 $P(X = a) = 0$입니다. 확률은 반드시 구간으로 계산합니다:

$$P(a \le X \le b) = \int_a^b f(x)\,dx$$
개념이산형↔연속형
확률 함수PMF: $P(X=x)$⇄PDF: $f(x)$
누적분포$F(x)=\sum_{t\le x}P(X=t)$⇄$F(x)=\int_{-\infty}^{x}f(t)dt$
기대값$\sum x P(X=x)$⇄$\int x f(x)dx$
분산$\sum(x-\mu)^2 P(x)$⇄$\int(x-\mu)^2 f(x)dx$
합 = 1$\sum P(X=x)=1$⇄$\int f(x)dx=1$
도구합(Σ sigma)⇄적분(∫ integral)
이산에서 연속으로 넘어갈 때 핵심 변화는?

핵심 변화 3가지:

  • 합(Σ) → 적분(∫): 개별 점을 더하는 대신 연속 곡선 아래 면적을 구함
  • PMF → PDF: "이 값일 확률"이 "이 근처의 확률 밀도"로 변환
  • P(X=a) > 0 → P(X=a) = 0: 연속형에서 정확히 한 점의 확률은 0. 구간의 면적만 의미 있음

하지만 기대값과 분산의 의미는 동일합니다. 도구(합 vs 적분)가 다를 뿐입니다.

5. 확률밀도 ≠ 확률

이것은 확률론에서 가장 많이 헷갈리는 개념입니다. 아래 세 가지 질문에 대한 답을 기억하세요:

질문답이유
y축의 밀도(Density)는 확률인가?아니다밀도는 1을 초과할 수 있음
확률과 밀도는 같은가?다르다밀도 x 구간폭 = 확률
밀도를 다 더하면 1인가?아니다적분하면 1 (더하면 아님)

$$\text{확률} = \int_a^b f(x)\,dx = \text{곡선 아래 면적} \quad \Big(\text{밀도 } f(x) \text{ 자체는 확률이 아님!}\Big)$$

위 탐색기에서 $\sigma$를 0.2로 줄여보세요. 최대 밀도가 약 2.0까지 올라갑니다!

이는 값이 매우 좁은 범위에 집중되어 있기 때문입니다. 밀도가 높아도 x축 구간이 좁으면 면적(확률)은 여전히 1 이하입니다.

비유: 빌딩이 아무리 높아도(밀도 큼), 바닥 면적이 작으면(구간 좁음) 총 부피(확률)는 제한됩니다.

히스토그램의 bins 수가 분포 모양을 바꾸는 이유

히스토그램은 데이터를 구간(bin)으로 나누어 빈도를 셉니다. 이때:

  • bins가 너무 적으면: 세부 패턴이 사라짐 (과소적합)
  • bins가 너무 많으면: 노이즈가 패턴처럼 보임 (과적합)
  • KDE(Kernel Density Estimation): 히스토그램의 이산적 한계를 극복하여 부드러운 연속 밀도 곡선을 추정

이는 ML의 편향-분산 트레이드오프와 같은 원리입니다.

6. ML에서의 기대값/분산

기대값과 분산은 머신러닝의 핵심 개념 곳곳에 등장합니다.

6.1 손실함수의 기대값

모델의 "평균적 성능"을 측정할 때 기대값을 사용합니다:

$$\text{Risk} = E[\mathcal{L}(Y, \hat{Y})] = \int \mathcal{L}(y, \hat{y}) \cdot p(x, y)\,dx\,dy$$

이것이 기대 위험(Expected Risk)으로, 모델이 새로운 데이터에서 평균적으로 얼마나 틀리는지를 나타냅니다. 우리가 실제로 최소화하려는 대상입니다.

6.2 편향-분산 트레이드오프 (Bias-Variance Tradeoff)

모델의 에러는 다음과 같이 분해됩니다:

$$E[(\hat{Y} - Y)^2] = \underbrace{\text{Bias}^2}_{\text{편향}^2} + \underbrace{\text{Variance}}_{\text{분산}} + \underbrace{\sigma^2}_{\text{노이즈}}$$

항목의미높으면?
편향(Bias)모델이 체계적으로 빗나가는 정도과소적합(Underfitting)
분산(Variance)데이터에 따라 예측이 흔들리는 정도과적합(Overfitting)
노이즈($\sigma^2$)데이터 자체의 불확실성줄일 수 없음
기대값/분산이 등장하는 ML 개념들
  • 정규화(Normalization): 평균 = 0, 분산 = 1로 스케일링 -> $z = \frac{x-\mu}{\sigma}$
  • Batch Normalization: 각 미니배치의 E[X], Var[X]를 계산하여 정규화
  • 확률적 경사 하강법(SGD): 전체 기울기의 기대값을 미니배치로 추정
  • VAE(Variational Autoencoder): 잠재 공간의 평균과 분산을 학습
  • 강화학습: 보상의 기대값 $E[R]$을 최대화
이 장의 핵심 요약
  • 기대값 E[X]: 확률로 가중 평균한 "무게 중심"
  • 분산 Var[X]: 기대값 주변의 "흩어짐" 정도
  • 이산 ↔ 연속: Σ ↔ ∫, PMF ↔ PDF (개념은 동일)
  • 밀도 ≠ 확률: 확률은 면적(적분), 밀도는 높이(1 초과 가능)
  • ML 활용: 손실함수, 편향-분산, 정규화 등 어디서나 기대값/분산 등장
