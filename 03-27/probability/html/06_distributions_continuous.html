1. 가우시안(정규) 분포란?

자연 현상에서 가장 흔하게 관찰되는 확률분포입니다. 같은 분포를 부르는 세 가지 이름이 있습니다.

용어 정리

이름유래사용 맥락
가우시안 분포
(Gaussian Distribution)		 수학자 Carl Friedrich Gauss의 이름 수학, 물리학, 공학에서 주로 사용
정규분포
(Normal Distribution)		 "정규(normal) = 표준적, 일반적"이라는 의미 통계학에서 가장 보편적인 이름
종형 곡선
(Bell Curve)		 그래프 모양이 종(bell)과 유사 일상 대화, 교육에서 직관적으로 사용
세 이름 모두 동일한 확률분포를 가리킵니다. 논문이나 교재에서 혼용되니, 셋 다 기억해두세요.

수식 (확률밀도함수, PDF)

여기서 $\mu$ = 평균(mean), $\sigma$ = 표준편차(standard deviation), $\sigma^2$ = 분산(variance)

수식이 복잡해 보이지만, 핵심은 딱 두 개의 파라미터입니다.

  • $\mu$ (뮤) : 분포의 중심 위치를 결정. 곡선을 좌우로 이동시킴
  • $\sigma$ (시그마) : 분포의 퍼짐 정도를 결정. 클수록 넓고 낮은 곡선
왜 정규분포가 중요한가? (ML 관점)
  • 중심극한정리(CLT): 표본 크기가 충분히 크면, 어떤 분포에서 추출하든 표본평균의 분포가 정규분포에 근사합니다. 이것이 정규분포가 "정규(normal)"인 이유입니다.
  • 많은 ML 알고리즘의 전제: 선형회귀의 잔차, Naive Bayes의 조건부 확률, GDA(Gaussian Discriminant Analysis) 등이 정규분포를 가정합니다.
  • 가우시안 노이즈: 센서 데이터, 이미지 노이즈, 통신 신호의 잡음은 대부분 정규분포를 따릅니다.

2. Galton Board — 정규분포가 자연스럽게 나타나는 이유

Galton Board(골턴 보드)는 영국 통계학자 Francis Galton이 1889년에 고안한 장치입니다. 구슬이 못(peg)에 부딪히며 좌우로 무작위 이동하면, 바닥에 쌓이는 모양이 자연스럽게 종형 곡선을 형성합니다.

이론: 베르누이 시행 → 이항분포 → 정규분포

Galton Board 각 층(Layer)의 의미

1
각 못(peg) = 1회 베르누이 시행
구슬이 못에 부딪힐 때, 왼쪽 또는 오른쪽으로 가는 것은 동전 던지기(성공/실패)와 동일합니다. 확률 $p$로 오른쪽, $1-p$로 왼쪽으로 이동합니다.
2
$n$개 층 통과 = $n$회 베르누이 시행의 합 = 이항분포 $B(n, p)$
구슬이 $n$개의 못을 거치며 오른쪽으로 간 횟수 $k$는 이항분포를 따릅니다. 바닥의 $k$번째 칸에 도달할 확률은 $P(X=k) = \binom{n}{k}p^k(1-p)^{n-k}$입니다.
3
$n$이 충분히 크면 → 정규분포에 근사
드 무아브르-라플라스 정리에 의해, $n$이 커지면 이항분포 $B(n,p)$는 평균 $\mu = np$, 분산 $\sigma^2 = np(1-p)$인 정규분포 $N(np, np(1-p))$에 수렴합니다. 이것이 Galton Board에서 구슬이 종형 곡선을 만드는 수학적 이유입니다.
동전 던지기 관점: 10개 동전을 한꺼번에 던지는 것을 149회 반복한다고 생각해보세요. 앞면(H)=+1, 뒷면(T)=-1로 놓고 합계를 구하면, 그 분포가 곧 Galton Board 바닥의 히스토그램입니다.
왜 Galton Board를 이해해야 하나? (ML 관점)
  • Random Walk: 주가 변동, 브라운 운동 등 현실의 많은 현상이 Galton Board와 동일한 원리(독립적 무작위 보폭의 합)를 따릅니다.
  • 중심극한정리의 시각적 증명: 개별 시행은 이산적(좌/우)이지만, 많은 시행의 합은 연속적 정규분포로 수렴합니다.
  • 딥러닝 가중치 초기화: He/Xavier 초기화가 정규분포를 사용하는 이유도 이 원리와 연결됩니다.

3. 원자료 → 표준화 (Standardization)

변환 3단계

원자료를 Z-score로 바꾸는 과정

1
원자료 (Raw Data): 측정값 그대로. 예) 시험 점수 [55, 70, 85, 90, 100]
단위와 스케일이 변수마다 다름
2
평균중심화 (Mean Centering): $X - \mu$
평균을 0으로 이동. 데이터의 위치만 바뀌고, 퍼짐 정도는 그대로.
3
표준화 (Standardization / Z-score 변환)
평균 = 0, 표준편차 = 1인 표준정규분포 $N(0, 1)$로 변환. 원래 단위가 사라지고, "평균에서 표준편차 몇 배만큼 떨어져 있는가"를 나타냄.

시각화: 원자료 → 평균중심화 → Z-score

아래 세 그래프는 같은 데이터가 변환 단계별로 어떻게 바뀌는지 보여줍니다. $\mu$와 $\sigma$ 슬라이더를 조절하여 다양한 원자료를 실험해보세요.

왜 표준화하는가?

표준화가 필요한 진짜 이유 (통계학 + ML 관점)

통계학 관점:

  • 비교 가능성: 국어 85점(평균 70, 표준편차 10)과 수학 90점(평균 85, 표준편차 15)을 직접 비교할 수 없지만, Z-score로 변환하면 "평균 대비 상대적 위치"로 공정하게 비교할 수 있습니다.
  • 확률 계산: 모든 정규분포를 $N(0,1)$로 통일하면, 하나의 표준정규분포표만으로 어떤 정규분포든 확률을 구할 수 있습니다.

머신러닝 관점:

  • 경사하강법 수렴 속도: 피처 스케일이 다르면 손실 함수의 등고선이 찌그러져 학습이 느려집니다. 표준화하면 등고선이 원형에 가까워져 빠르게 수렴합니다.
  • 거리 기반 알고리즘: KNN, SVM, K-Means 등은 거리를 계산하므로, 스케일이 큰 변수가 결과를 지배하는 것을 방지합니다.
  • 정규화 효과: 가중치의 크기가 비슷해져 L1/L2 정규화가 모든 피처에 공평하게 적용됩니다.

4. 표준정규분포와 분포표

표준정규분포란 $\mu = 0$, $\sigma = 1$인 특수한 정규분포입니다. 기호로 $N(0, 1)$이라 쓰고, 확률변수는 관례적으로 $Z$로 표기합니다.

$N(0,1)$의 확률밀도함수 — $\mu=0$, $\sigma=1$을 대입한 결과

$\sigma$-구간 확률 (68-95-99.7 법칙)

정규분포에서 데이터가 평균으로부터 일정 표준편차 이내에 들어갈 확률은 다음과 같습니다.

구간확률의미
$\mu \pm 1\sigma$68.27%데이터의 약 2/3가 이 범위
$\mu \pm 2\sigma$95.45%거의 모든 데이터
$\mu \pm 3\sigma$99.73%이탈치(outlier) 기준
68-95-99.7 법칙이 ML에서 왜 중요한가?
  • 이상치 탐지: Z-score > 3이면 99.7% 밖의 극단값. 데이터 전처리에서 이상치 필터링에 직접 사용됩니다.
  • 신뢰구간: 모델 예측의 불확실성을 "평균 ± 1.96σ = 95% 신뢰구간"으로 표현합니다.
  • A/B 테스트: p-value가 0.05 미만인지(= 95% 신뢰수준) 판정할 때 이 확률값을 사용합니다.

$P(-1 < Z \le 1) = 0.68$ 유도

68%의 산술식

1
표준정규분포의 누적분포함수(CDF)를 $\Phi(z)$라 하면:
$\Phi(z) = P(Z \le z)$
2
분포표에서 $\Phi(1) = P(Z \le 1) = 0.8413$
3
대칭성에 의해: $\Phi(-1) = P(Z \le -1) = 1 - \Phi(1) = 1 - 0.8413 = 0.1587$
4
구간 확률 계산:
$P(-1 < Z \le 1) = \Phi(1) - \Phi(-1) = 0.8413 - 0.1587 = \mathbf{0.6826} \approx 68\%$

$P(-1.96 < Z \le 1.96) = 0.95$ 유도

95%의 산술식 (신뢰구간의 기초)

1
분포표에서 $\Phi(1.96) = P(Z \le 1.96) = 0.9750$
2
오른쪽 꼬리: $P(Z > 1.96) = 1 - 0.9750 = 0.0250$
3
대칭성: $\Phi(-1.96) = 1 - 0.9750 = 0.0250$
4
$P(-1.96 < Z \le 1.96) = \Phi(1.96) - \Phi(-1.96) = 0.9750 - 0.0250 = \mathbf{0.9500} = 95\%$
1.96이라는 숫자가 통계학에서 유독 자주 등장하는 이유가 바로 이것입니다. "양쪽 꼬리에 각각 2.5%씩 남기면 중앙 95%"라는 의미이며, 95% 신뢰구간 = $\bar{X} \pm 1.96 \times \frac{\sigma}{\sqrt{n}}$의 1.96이 여기서 나옵니다.
