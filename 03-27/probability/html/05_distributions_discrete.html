1. 확률분포가 뭐길래?

"표본의 확률분포를 안다는 것은 표본 그 자체를 넘어 모집단을 이해하는 열쇠다."

데이터를 수집하면 우리는 표본을 얻는다. 그런데 진짜 알고 싶은 건 모집단 전체의 성질이다. 표본이 어떤 모수(Parameter)를 가지는 분포를 따르는지 알면, 예측 / 추론 / 의사결정이 가능해진다.

왜 확률분포를 배워야 하나?
  • 예측: 내일 불량품이 몇 개 나올까? → 이항분포 / 포아송분포
  • 추론: 이 동전이 공정한가? → 가설 검정 (베르누이 기반)
  • 의사결정: A/B 테스트에서 어느 안이 나은가? → 분포 비교

분포를 알면 확률의 언어로 세상을 기술할 수 있다.

베르누이분포 (Bernoulli Distribution)

정의: 결과가 성공(1) 또는 실패(0) 두 가지뿐인 단일 시행

모수: p (성공확률, 0 < p < 1)

예시: 동전 1회 던지기, 합격/불합격 판정

관계: 이항분포에서 n=1인 특수한 경우

2. 베르누이 시행 (Bernoulli Trial)

핵심 용어

용어의미예시
베르누이 시행결과가 딱 2가지(성공/실패)인 하나의 실험동전 1회 던지기
성공확률 p성공이 나올 확률공정한 동전: p = 0.5
실패확률 qq = 1 - p공정한 동전: q = 0.5
확률변수 XX = 1(성공) 또는 X = 0(실패)앞면=1, 뒷면=0
베르누이분포 PMF P(X = k) = pk(1-p)1-k,   k ∈ {0, 1}

평균: E[X] = p  |  분산: Var(X) = p(1-p)

찌그러진 동전이란?

공정한 동전은 p = 0.5이지만, 현실에서는 p ≠ 0.5인 경우가 많다.

  • p = 0.7인 동전: 앞면이 70% 확률로 나오는 "찌그러진" 동전
  • 과제: p = 0.7, p = 0.3 각각에 대해 베르누이/이항분포 실험

3. 이항분포 (Binomial Distribution)

베르누이 시행을 n번 독립 반복했을 때, 성공 횟수 X의 분포가 바로 이항분포다.

이항분포 PMF (확률질량함수) P(X = k) = C(n, k) · pk · (1-p)n-k,   k = 0, 1, ..., n

C(n, k) = n! / (k!(n-k)!) — "n개에서 k개 고르는 조합의 수"

통계량공식의미
평균 (기댓값)E[X] = npn번 시행 시 평균 성공 횟수
분산Var(X) = np(1-p)성공 횟수의 흩어진 정도
표준편차σ = √(np(1-p))평균에서 벗어나는 정도
동전 100번 던지면 앞면이 정확히 50번?

"동전 100번 던져서 앞면이 몇 번?" → 55번? 45번? 50번? — 케바케(케이스 바이 케이스)!

하지만 이 시행을 10,000번 반복하면 성공 횟수의 분포가 이항분포를 따른다.

4. 이항분포 → 정규분포

이항분포에서 n이 충분히 크면 (일반적으로 n ≥ 30), 분포의 모양이 종 모양(bell shape)의 정규분포에 점점 가까워진다. 이것이 중심극한정리(CLT)의 핵심 아이디어이다.

정규근사 (Normal Approximation) n이 크면:   X ~ Bin(n, p)   ≈   N(np, np(1-p))
왜 중요한가?

이항분포의 정확한 계산은 n이 클 때 매우 번거롭다. 하지만 정규분포로 근사하면:

  • z-score를 이용한 빠른 확률 계산 가능
  • 모든 통계적 추론(구간 추정, 가설 검정)에 정규분포 활용 가능
  • "이산 → 연속"으로의 다리 역할

  • 시뮬레이션 평균 ≈ 이론적 평균 (np)
  • 시뮬레이션 표준편차 ≈ 이론적 표준편차 (√(np(1-p)))
  • 빨간 곡선(이론적 정규분포)과 히스토그램이 거의 겹침
  • p = 0.5에서 가장 대칭적, p가 0이나 1에 가까우면 비대칭

5. Python 과제 가이드

조상구 교수님 과제에서 사용할 핵심 코드 패턴:

5-1. 베르누이 시행 & 이항분포 (scipy.stats)

from scipy.stats import bernoulli, binom
import numpy as np
import matplotlib.pyplot as plt

# === 베르누이 시행 ===
p = 0.5                      # 성공확률
samples = bernoulli.rvs(p, size=1000)  # 1000번 시행
print(f"성공 비율: {samples.mean():.3f}")

# === 이항분포 (N=10) ===
n, p = 10, 0.5
x = np.arange(0, n+1)
pmf = binom.pmf(x, n, p)     # PMF 계산

plt.bar(x, pmf, color='steelblue', alpha=0.8)
plt.xlabel('성공 횟수 (k)')
plt.ylabel('P(X=k)')
plt.title(f'이항분포 B({n}, {p})')
plt.show()

5-2. 찌그러진 동전 (p=0.7)

# 찌그러진 동전: p=0.7
p_biased = 0.7
n = 10

# 베르누이 시행 1000번
bern_samples = bernoulli.rvs(p_biased, size=1000)

# 이항분포 시뮬레이션
binom_samples = binom.rvs(n, p_biased, size=10000)

fig, axes = plt.subplots(1, 2, figsize=(12, 5))

# 왼쪽: 베르누이
axes[0].hist(bern_samples, bins=2, edgecolor='black')
axes[0].set_title(f'베르누이(p={p_biased})')

# 오른쪽: 이항분포
axes[1].hist(binom_samples, bins=range(n+2), edgecolor='black', density=True)
axes[1].set_title(f'이항분포 B({n}, {p_biased})')
plt.tight_layout()
plt.show()

5-3. 이항분포 → 정규분포 수렴 확인

from scipy.stats import norm

n_values = [10, 30, 100, 1000]
p = 0.5

fig, axes = plt.subplots(2, 2, figsize=(12, 10))
for ax, n in zip(axes.flat, n_values):
    x = np.arange(0, n+1)
    pmf = binom.pmf(x, n, p)
    ax.bar(x, pmf, alpha=0.6, label='이항분포')

    # 정규근사 오버레이
    mu, sigma = n*p, np.sqrt(n*p*(1-p))
    x_cont = np.linspace(mu - 4*sigma, mu + 4*sigma, 200)
    ax.plot(x_cont, norm.pdf(x_cont, mu, sigma),
           'r-', lw=2, label='정규근사')
    ax.set_title(f'n={n}, p={p}')
    ax.legend()

plt.suptitle('n이 커질수록 정규분포에 수렴', fontsize=14)
plt.tight_layout()
plt.show()
